ModalCSS Cloudinary upload

A javascript functionality which helps you to accelerate the integration of Cloudinary in any new Vanilla JS project.
This sample works upon the CSS3 modal and Pure CSS.

The uploaded file url will be "saved" to the "cloudinaryLoadedImg" hidden input. Based on the "pure JS Cloudinary sample" at codepen

Cloudinary upload

Upload multiple files using the select button or by doing drag and drop within the dashed area.
